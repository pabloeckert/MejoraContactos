# 🤝 Contribuir a MejoraContactos

¡Gracias por tu interés en contribuir!

## Desarrollo

```bash
git clone https://github.com/pabloeckert/MejoraContactos.git
cd MejoraContactos
npm install --legacy-peer-deps
npm run dev
```

## Convenciones

- **Commits:** `tipo: descripción` (feat, fix, docs, chore, ci, perf)
- **Branches:** `feature/nombre`, `fix/nombre`, `docs/nombre`
- **Tests:** Todos los PRs deben pasar `npm test` (199+ tests)
- **Lint:** `npm run lint` debe pasar sin errores

## Pull Requests

1. Fork el repo
2. Crear una branch desde `main`
3. Hacer cambios
4. Asegurar que `npm test` pasa
5. Abrir PR con descripción clara

## Áreas de contribución

- 🐛 Bug fixes
- ✨ Nuevas funcionalidades
- 📝 Documentación
- 🌍 Traducciones (i18n)
- 🧪 Tests
- ⚡ Performance

## Stack

React 18 + Vite + TypeScript + Tailwind + shadcn/ui + Supabase Edge Functions
